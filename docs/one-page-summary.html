GA 프로모션ON 1페이지 요약

월별 프로모션 총괄판 엑셀을 업로드하면 지점별 현황과 대리점별 최종 시책안내장 JPG를 하나의 공유 URL에서 확인·다운로드하는 업무 효율화 PoC.

핵심 목표

카톡 파일 반복 첨부와 수작업 장표 재가공을 줄이고, RM·영업추진P·GA사업부 프로모션 담당자가 확정값을 한 번만 올리면 현장은 같은 URL에서 최신 확정본을 확인하게 한다.

문제 정의

  • GA별 조건 차이로 프로모션 재가공 업무가 반복된다.
  • 카톡 파일 공유는 최신본·대상자·확정 여부 혼선을 만든다.
  • 현장 문의와 지점장 추가 업무가 반복 발생한다.

주요 사용자

  • Primary: RM, 영업추진P, GA사업부 프로모션 담당자.
  • Secondary: 지점장, GA영업관리자.
  • 운영 검토자: 승인·노출·다운로드 기준 관리.

운영 기준

  • 한 화면은 그 달의 최종 장표 1장 기준.
  • 협의중은 숫자·JPG 제한, 확정은 미리보기·다운로드 허용.
  • 일반 사용자는 승인권한 없이 확정본만 확인.

입력 → 처리 → 출력

Input

월별 엑셀 총괄판, 대리점명/코드, 담당 지점, 기준월, 상품군별 지급률, 기존 포스터 배경

Process

엑셀 업로드 → 컬럼·숫자 검증 → 지점/대리점별 정규화 → 협의중/확정 판정 → 장표 렌더링

Output

단일 공유 URL, 지점별 대시보드, 대리점별 포스터형 화면, 최종 JPG 다운로드

핵심 기능

엑셀 총괄판 업로드 지점별 현황 자동 반영 월/대리점/상태 필터 최종 JPG 미리보기 JPG 다운로드 PC/모바일 자동 전환

보류 범위

  • 협상 판단 자동화.
  • 상품전략·지점장 취향 커스터마이징.
  • 외부 사용자 무제한 조회.

완료 정의 더미 또는 월별 총괄판 엑셀로 지점별 현황이 자동 반영되고, 대리점별 JPG 미리보기·다운로드가 가능하며, 업로드·생성·다운로드 테스트 10회가 오류 없이 수행되면 v1 완료.
