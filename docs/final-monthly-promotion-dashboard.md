# 최종 방향 정리

## 방향 변경
- 최종 목표는 하나의 홈페이지형 공유 URL에서 `그 달의 대리점별 프로모션 장표 1장`을 확인하고 JPG 파일로 바로 다운로드할 수 있게 만드는 것이다.
- 기본 대시보드는 기존 포스터 배경·디자인 양식을 바탕으로 하되, 숫자는 엑셀 총판 업로드 값을 자동 반영한다.
- 월별 전체 이력보다 `이번 달 확정값`, `공유 홈페이지 목록`, `승인 상태`, `대리점별 JPG 생성 여부`가 더 중요하다.

## 최종 출력 기준
1. 입력: 월별 엑셀 총판, 대리점명/코드, 협의상태, 시상값, 상품군별 지급률
2. 처리: 필수 컬럼 검증, 숫자 형식 확인, 대리점별 데이터 DB화, 템플릿 숫자 주입
3. 승인: 메인 유저가 확정 수치와 최종 JPG 미리보기를 검토
4. 출력: 하나의 공유 홈페이지에서 프로모션 목록 확인 및 대리점별 최종 시책안내장 JPG 다운로드

## 구현 원칙
- 한 홈페이지 안에서 월/대리점/상태별로 필요한 프로모션을 찾을 수 있게 한다.
- 협의중 데이터는 숫자와 JPG 다운로드를 제한한다.
- 확정/승인 완료 건만 JPG 생성 및 다운로드를 허용한다.
- JPG 파일명은 기준월, 대리점명, 확정본 여부를 포함해 최신본 혼선을 줄인다.
