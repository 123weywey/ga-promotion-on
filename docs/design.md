# design.md — 화면 규칙

> 최종 산출물은 `그 달의 대리점별 시책안내장 JPG`다. 대시보드는 설명용 랜딩페이지가 아니라 하나의 홈페이지형 공유 URL에서 월별·대리점별 프로모션을 확인하고 JPG를 다운로드하는 운영 화면이어야 한다.

- 폰트: 맑은 고딕 또는 가독성 높은 산세리프, 숫자는 굵게 표시해 시상값이 즉시 보이게 한다.
- 색: 밝은 회색/흰색 바탕에 삼성 블루 계열을 기준색으로 쓰고, `확정`은 녹색, `협의중`은 주황, `오류`는 붉은색으로 구분한다.
- 구조: 홈페이지 공유 URL → 월/대리점 검색·필터 → 엑셀 업로드 상태 → 승인 대기/완료 → JPG 생성 현황 → 대리점별 다운로드 목록 → 최종 장표 미리보기 순서로 둔다.
- 최종 장표: 기존 포스터 배경·디자인 양식을 유지하고, 엑셀 총판의 숫자만 지정 영역에 자동 주입한다.
- 상태: 업로드 전 / 검증 오류 / 협의중(숫자 비공개) / 승인 대기 / 확정·JPG 생성 완료 / 다운로드 완료.
- URL 공유: 카톡에는 파일 여러 개 대신 홈페이지 링크 하나만 보내고, 사용자는 같은 페이지에서 월/대리점/상태별 프로모션을 찾아 JPG를 받는다.
- 원칙: 월별 이력 비교를 길게 보여주지 않고, `이번 달 최종본`과 `대리점별 JPG 다운로드 가능 여부`를 먼저 보여준다.
