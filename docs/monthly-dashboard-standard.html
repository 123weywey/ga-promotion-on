표준 대시보드 핵심 표

5월 총괄판에서 지점장이 바로 보는 데 유용한 컬럼을 중심으로 표준화한 예시입니다.

영역 표준 컬럼 설명
기본 식별 월, 지점명, 코드, GA명 월별 전개 기준과 조회 대상을 명확히 구분
대상 여부 지점별 대상여부, 담당 지점 해당 지점이 실제 전개 대상인지 즉시 판단
프로모션 유형 순수3종 유형, 행복종신 유형 대리점별 적용 타입을 한 줄로 확인
핵심 지급률 5년 익월, 7년 익월, 플러스원 익월, 13회 지급률 지점장이 가장 자주 묻는 핵심 숫자를 압축 노출
부가 운영값 기본, 자체 선지급, 지사장, 블루멤버스, 브릿지 부가 시상과 지점 운영 포인트를 함께 확인
실행 지원 프로모션 담당, 마케팅 연계, 포스터/JPG 링크 조회 후 바로 전개 가능한 연결 지점

월 선택 후 상단 카드

1. 월 기본 정보

`26.5월 프로모션 안내`처럼 월 기준 제목과 최신 업데이트일을 상단에 고정합니다.

2. 장표 준비 상태

공문, 13회차, 총괄판, 환급률표, 포스터가 준비되었는지 배지로 표시합니다.

3. 핵심 지급률 요약

익월 지원, 13회차 지원, 대표 상품군, 순수형 안(400/100, 300/200)을 카드로 요약합니다.

4. 지점별 대상 바로 검색

지점명 입력 시 총괄판 기준으로 대상여부와 GA별 전개 유형을 바로 볼 수 있게 설계합니다.

공문 요약 카드

표준 제목

`삼성생명 26.X월 프로모션 안내`를 그대로 사용해 월 기준 혼선을 줄입니다.

핵심 필드

상품 구분, 상품 상세, 주보험 납기, 익월 지원, 13회차 지원, 비고를 고정 표시합니다.

건강상해 13회차 카드

표준 제목

`건강상해 13회차 프로모션`을 별도 카드로 분리해 메인 공문과 구분합니다.

핵심 필드

상품 상세, 주보험 납기, 13회차, GA지급률, 출시 예정/비고를 보여줍니다.

지점장 실행 카드

바로 전개

대리점용 공문, FC용 공문, 포스터, 총괄판 링크를 한 구역에 모아 클릭 동선을 줄입니다.

운영 메모

해당 월의 특이사항, 출시 예정 상품, 예외 지급률, 추가 전개 메모를 별도 박스로 둡니다.

장표를 찾는 시간을 줄이기 위해, 지점장에게 필요한 파일만 아래 순서대로 고정하는 것을 추천합니다.

포스터 26.5월 포스터 1P

운영상 최종 권장 구조

상단

`월 선택 + 준비 상태 + 핵심 지급률`만 먼저 보여줘서 지점장이 10초 안에 상황을 파악하게 합니다.

중단

`지점별 대상 현황 표`를 두고, 검색 또는 필터로 자기 지점 대상만 좁혀서 보게 합니다.

하단

`공문/포스터/JPG/총괄판` 바로가기를 두어 조회 후 바로 실행 단계로 넘어가게 합니다.

26.6월 처리 원칙

현재 확보된 26.6월 파일은 이전 월 zip을 담은 번들 보관본으로 확인되었습니다. 표준 대시보드는 26.1~26.5월 공통 구조로 먼저 만들고, 26.6월 상세 장표 확보 후 같은 틀에 추가하는 것이 안전합니다.
