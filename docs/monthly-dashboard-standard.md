# 월별 자료 구조 참고 메모

> 이 문서는 26.1월~26.6월 자료를 보고 공통 입력 항목을 정리한 참고 자료다. 최종 화면 기준은 `docs/final-monthly-promotion-dashboard.html`의 엑셀 총판 업로드 및 대리점별 JPG 다운로드 흐름이다.

## 분석 결론
- 26.1월부터 26.5월까지는 `메인 공문`, `FC용/대리점용 공문`, `건강상해 13회차`, `총괄판`, `환급률표 또는 환급률자료`, `포스터` 구조가 반복된다.
- 최종 PoC에서는 월별 비교판을 만들기보다, 해당 월 총판을 업로드해 대리점별 최종 안내장 JPG를 생성하는 구조가 더 효율적이다.
- 지점장 관점의 핵심은 `이번 달 확정본`, `대리점명`, `프로모션 유형`, `익월/13회 지급률`, `다운로드 가능한 JPG`다.

## 문서 근거
- 메인 공문 공통 제목: `삼성생명 '26年 X月 프로모션 안내`
- 건강상해 공통 제목: `삼성생명 '26年 X月 건강상해 13회차 프로모션 안내`
- 5월 총괄판 첫 시트 확인 항목: `지점별 대상여부`, `GA명`, `순수3종 프로모션 유형`, `행복종신 유형`, `5년/7년 익월`, `13회 지급률`, `담당`
