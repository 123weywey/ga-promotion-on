운영 흐름

문서에 있는 TO-BE 흐름을 지점장 확인 중심으로 재배열했습니다.

1. RM 상태 업데이트 대리점별 상태를 `협의중` 또는 `확정`으로 단순하게 관리합니다.
2. 지점장 검색 조회 지점장은 대리점명 검색만으로 최신 상태를 바로 확인합니다.
3. 확정 시 숫자 자동 반영 상단 시상값과 상품별 퍼센트가 포스터 템플릿에 자동 주입됩니다.
4. JPG 자동 생성 같은 데이터로 발표·공유용 이미지 산출물을 즉시 내려받을 수 있게 설계합니다.

입력

RM 입력값

진행상태, 기준월, 익월 시상, 자체 시상, 상품별 지급률 값을 관리합니다.

기준정보

대리점 코드와 한글 대리점명은 내부 기준정보로 유지하고, 화면에는 한글명만 노출합니다.

처리

상태 판정

`협의중`이면 숫자를 숨기고, `확정`이면 숫자를 공개하는 간단한 규칙으로 동작합니다.

템플릿 주입

고정 포스터 레이아웃에 변동값만 채워 넣어 일관된 결과물을 만듭니다.

출력

지점장 조회 화면

별도 학습 없이 검색 중심으로 최신 상태와 확정 수치를 확인할 수 있습니다.

공유용 산출물

포스터와 JPG를 같은 데이터로 생성해 최신본 기준을 하나로 맞춥니다.

AS-IS / TO-BE 비교 이미지

이전 작업에서 만든 비교 차트를 그대로 연결했습니다.

안전 및 운영 규칙

내부 사용자 중심

외부 경쟁사나 타 원수사 직원은 조회할 수 없도록 접근 통제가 필요합니다.

상태 우선 공개

협의중 단계에서는 확정되지 않은 숫자를 노출하지 않고, 대기와 오해를 줄이는 방향으로 운영합니다.

운영 단순성

상태값 2개와 고정 템플릿 중심으로 유지해 일반 직원도 수정·재사용할 수 있게 설계합니다.
