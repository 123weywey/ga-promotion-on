# GA 프로모션ON 1페이지 요약

## 프로젝트 한 줄 정의
`GA 프로모션ON`은 월별 프로모션 총괄판 엑셀을 업로드하면 지점별 확정/협의중 현황과 대리점별 최종 시책안내장 JPG를 하나의 공유 URL에서 확인·다운로드할 수 있게 만드는 GA사업부 업무 효율화 PoC다.

## 문제와 목표
- 문제: GA별 프로모션 조건 차이, 반복 장표 재가공, 카톡 파일 첨부 방식 때문에 최신본 혼선·현장 문의·지점장 추가업무가 반복된다.
- 목표: 엑셀 총괄판을 단일 입력원으로 삼아 지점별 현황과 확정 안내장 JPG를 자동 생성하고, 카톡에는 파일이 아니라 URL 하나만 공유한다.
- 우선순위: 업무시간 절감, 반복 입력·수작업 감소, 현장 문의 감소, 일반 직원도 운영 가능한 단순 구조.

## 주요 사용자
- Primary: RM, 영업추진P, GA사업부 프로모션 담당자.
- Secondary: 지점장, GA영업관리자.
- 운영 검토자: 확정 전 수치 노출, 승인권한, 최신본 배포 기준을 관리하는 보안·운영 담당자.

## 입력 → 처리 → 출력
| 구분 | 내용 |
|---|---|
| 입력 | 월별 엑셀 총괄판, 대리점명/코드, 담당 지점, 협의상태, 기준월, 상품군별 지급률, 기존 포스터 배경·디자인 양식 |
| 처리 | 엑셀 업로드 → 필수 컬럼·숫자 검증 → 지점/대리점별 데이터 정규화 → 협의중/확정 판정 → 대시보드 반영 → 포스터형 템플릿에 숫자 주입 → JPG 생성 |
| 출력 | 단일 홈페이지형 공유 URL, 지점별 프로모션 현황 대시보드, 대리점별 최종 포스터형 화면, 최종 JPG 다운로드 |

## 핵심 기능
- 관리자모드에서 월별 프로모션 총괄판 `.xlsx` 업로드 후 지점별 총 지사, 확정 완료, 협의중, 대리점 목록 자동 반영.
- 사용자가 지점명·월·상태를 선택하면 담당 지사의 확정/협의중 프로모션을 한 화면에서 확인.
- 확정 건은 미리보기와 JPG 다운로드를 활성화하고, 협의중 건은 상태만 표시해 숫자와 다운로드를 제한.
- PC/모바일에서 같은 URL을 열면 화면 크기에 맞춰 자동 전환.

## 운영·승인·보안 원칙
- 카톡에는 개별 파일을 반복 첨부하지 않고 공유 URL 1개만 배포한다.
- 메인 유저가 업로드 결과, 확정 수치, 홈페이지 노출, 최종 JPG를 검토·승인한다.
- 일반 사용자는 승인권한 없이 확정본 조회·다운로드만 가능하게 한다.
- 확정 전 수치는 비공개 또는 제한하고, JPG 파일명은 기준월·대리점명·생성일을 포함한다.

## 보류 범위
- 협상 판단 자동화, 상품전략 반영, 지점별 취향 커스터마이징, 외부 사용자 무제한 조회는 v1 범위에서 제외한다.
- 초기 버전은 정확한 월별 프로모션 안내장 JPG 구현과 공유 URL 기반 현장 배포 안정화에 집중한다.

## 완료 정의
더미 또는 월별 총괄판 엑셀로 지점별 현황이 자동 반영되고, 대리점별 시책안내장 JPG 미리보기·다운로드가 가능하며, 업로드·생성·다운로드 테스트 10회가 오류 없이 수행되면 v1 완료로 본다.
