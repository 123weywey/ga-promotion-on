# 프로모션 ON

GA사업부 프로모션 공유를 위해 만든 단일 대시보드형 운영 화면입니다.

## 배포 URL

- 운영 URL: https://ga-promotion-on.vercel.app/docs/final-monthly-promotion-dashboard

## 제품 소개

프로모션 ON은 월별 `프로모션 총괄판`, `점포현황`, `PPT 템플릿` 3개 파일을 기준으로 지점별 프로모션 현황을 한 화면에서 조회하고, 최종 프로모션 미리보기와 JPG/PPT 다운로드까지 연결하는 GA사업부용 공유 대시보드입니다.

- 관리자 모드에서 월별 운영 파일 3종을 업로드하면 Supabase 기준 데이터가 갱신됩니다.
- 사용자는 지역단/지점/상태/대리점 검색으로 필요한 대상만 빠르게 확인할 수 있습니다.
- 휴대폰과 PC 모두 동일한 공통 URL에서 같은 대시보드를 사용합니다.
- 월별 업로드 원본 파일(.xlsx, .pptx)과 업로드 기반 생성 데이터(data/)는 GitHub에 저장하지 않습니다.
- 운영 데이터는 Supabase 저장소를 기준으로 관리합니다.
